# Workflows with Nuxt (Nitro v2)

Learn more about Nuxt: https://nuxt.com

## Commands

**Local development:**

```sh
npm run dev
```

**Production build (Vercel):**

```sh
NITRO_PRESET=vercel npm run build
npx vercel --prebuilt
```

**Production build (Node.js):**

```sh
npm run build
node .output/server/index.mjs
```
